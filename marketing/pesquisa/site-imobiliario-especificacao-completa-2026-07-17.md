## 1. CONTEXTO DO PROJETO

### 1.1 Objetivo Principal
Produzir um site imobiliário de alto padrão para captar leads qualificados e vender imóveis em **Itajaí** e **Balneário Camboriú** (SC) via tráfego pago, com meta de **primeira venda no primeiro mês**.

### 1.2 Cliente
- **Segmento:** Imobiliária / Corretor de imóveis de alto padrão
- **Região:** Litoral norte de Santa Catarina
- **Foco:** Empreendimentos residenciais (lançamentos, na planta, prontos e de luxo)
- **Diferencial:** Tráfego pago otimizado + site que gera desejo e conversão

### 1.3 Região de Atuação

| Cidade | Destaque | Preço Médio/m² | VGV 2025 |
|---|---|---|---|
| **Itajaí** | Crescimento de 46% no VGV em 3 anos | R$ 12.848 | R$ 3,2B+ |
| **Balneário Camboriú** | 5ª maior força imobiliária do Brasil | R$ 15.000+ | R$ 8B+ |
| **Itapema** | Expansão rápida, nova Praia Brava | R$ 11.000 | R$ 2B+ |

**Mercado total do litoral norte SC:** R$ 13,47 bilhões em VGV (2025)

### 1.4 Bairros Prioritários

**Itajaí:**
- Fazenda (consolidado, valorização)
- Centro (alta demanda de busca, porto)
- Cordeiros (em valorização)
- São Vicente (familiar, acessível)
- Dom Bosco (familiar, lançamentos)
- São João (micro-apartamentos)
- Praia Brava (nobre, em ascensão)

**Balneário Camboriú:**
- Barra Sul (marina, luxo)
- Centro (consolidado, alto padrão)

### 1.5 Perfil dos Compradores

| Segmento | % do Mercado | Origem | Motivação | Ticket |
|---|---|---|---|---|
| Agronegócio | 30% | MT, PR, MG, RS | Ativo estratégico, diversificação | R$ 1M-5M |

### 1.6 Diferencial do Projeto vs Concorrentes

## 4. 10 BRECHAS DE MERCADO

### Brecha #1: Site em Inglês
- **Problema:** 10% das vendas do Senna Tower foram para estrangeiros, mas nenhuma das 3 imobiliárias líderes tem site em inglês
- **Oportunidade:** Capturar demanda internacional (Golden Visa, VIPER, lifestyle)
- **Implementação:** Página /en/ mirror + /foreign-buyers/ com processo passo-a-passo
- **Investimento:** R$ 5-10K (tradução + adaptação)
- **Retorno:** 5-15% aumento de leads qualificados

### Brecha #2: Tour Virtual 360
- **Problema:** Nenhuma imobiliária em BC oferece tour virtual 360
- **Custo:** R$ 250-400 por tour ou R$ 1.200/mês pacote (50 imóveis = R$ 15-20K/mês)
- **Oportunidade:** Diferencial enorme, reduz visitas físicas, aumenta conversão
- **Implementação:** Embed em cada empreendimento + link WhatsApp + Google Meu Negócio
- **Retorno:** +25% taxa de visita, +15% conversão

### Brecha #3: Conteúdo Educativo (Blog SEO)
- **Problema:** Nenhuma imobiliária tem blog com SEO estratégico
- **Oportunidade:** Tráfego orgânico gratuito, autoridade, backlinks
- **Tópicos sugeridos:**
  - "Por que Itajaí valorizou 46% em 2025"
  - "Como investir em imóveis no litoral de SC"
  - "Airbnb em Balneário Camboriú: guia completo"
  - "Golden Visa Brasil: imóveis de R$ 1M+"
  - "ITBI em BC: quanto custa e como calcular"
- **Investimento:** R$ 2-5K/mês (redação + SEO)
- **Retorno:** Tráfego orgânico crescente, autoridade de domínio

### Brecha #4: Experiência Mobile
- **Problema:** Sites dos concorrentes são lentos, não otimizados para mobile
- **Oportunidade:** 70%+ do tráfego imobiliário vem de mobile
- **Implementação:** Core Web Vitals otimizado, AMP, PWA
- **Métricas alvo:** LCP < 2.5s, FID < 100ms, CLS < 0.1
- **Retorno:** +20% taxa de conversão mobile

### Brecha #5: LinkedIn B2B
- **Problema:** 75% das construtoras não têm LinkedIn ativo
- **Oportunidade:** Corretores de SP, RJ, MG que indicam clientes para BC estão no LinkedIn
- **Implementação:** Página /corretores-parceiros/ + programa de indicação
- **Programa:** "Indique seu cliente de R$ 2M+, ganhe 20% da comissão"
- **Conteúdo:** 3 posts/semana sobre mercado, oportunidades, cases
- **Retorno:** 10-30% dos leads de alto padrão via indicação

### Brecha #6: Calculadoras
- **Problema:** Mercedes Imóveis (Itajaí) tem 3 calculadoras, mas em BC nenhuma imobiliária tem
- **Oportunidade:** Ferramenta de conversão poderosa, gera leads qualificados
- **Calculadoras:**
  1. **Financiamento:** SAC + Price + IOF + seguro
  2. **ITBI:** Por município (BC = 3%, Itajaí = 2%, outros)
  3. **ROI Aluguel:** Anual vs temporada (Airbnb)
- **Investimento:** R$ 5-15K (desenvolvimento)
- **Retorno:** +30% leads qualificados, maior tempo no site

### Brecha #7: Retargeting Avançado
- **Problema:** Pouco uso de remarketing sequencial
- **Oportunidade:** Nutrir leads que visitaram empreendimentos específicos
- **Estratégia:**
  - Dia 1: Visitou empreendimento — anúncio com foto do empreendimento
  - Dia 3: Não converteu — vídeo de tour virtual
  - Dia 7: Não converteu — depoimento de comprador
  - Dia 14: Não converteu — oferta especial (desconto, condição)
  - Dia 30: Não converteu — convite para visita presencial
- **Retorno:** +40% conversão de leads existentes

### Brecha #8: Marca Institucional vs Fundador
- **Problema:** Todos os concorrentes focam no fundador, não na empresa
- **Risco:** Se o fundador sair, a marca perde valor
- **Oportunidade:** Criar marca institucional blindada
- **Implementação:** Site com processo documentado, equipe apresentada, cases de sucesso
- **Retorno:** Valorização da empresa, facilidade de venda/scaling

### Brecha #9: Parceria Hotelaria / Beach Clubs
- **Problema:** Nenhuma imobiliária integra lifestyle no site
- **Oportunidade:** BC é sinônimo de lifestyle, praia, luxo
- **Implementação:** Seção /parcerias/ com beach clubs, restaurantes, spas
- **Parceiros:** Beach clubs (P12, Warung), restaurantes (Madero, Coco Bambu), spas
- **Retorno:** Diferencial de marca, conteúdo para anúncios, parcerias comerciais

### Brecha #10: Dashboard Público de Mercado
- **Problema:** Nenhuma imobiliária em BC traduz dados FipeZap em dashboard interativo
- **Oportunidade:** Autoridade, backlinks, tráfego orgânico, PR
- **Implementação:** Página /bc-real-estate-index/
- **Dados:**
  - Preço médio/m² por bairro
  - Evolução 12 meses
  - Comparativo BC vs Itapema vs São Paulo
  - Volume de vendas por trimestre
  - Tempo médio de venda
- **Investimento:** R$ 10-20K (desenvolvimento + API FipeZap)
- **Retorno:** Autoridade de domínio, backlinks de mídia, tráfego orgânico

```typescript
// app/api/leads/route.ts
import { NextRequest, NextResponse } from "next/server";
import { Lead } from "@/lib/types";

export async function POST(request: NextRequest) {
  try {
    const body: Lead = await request.json();

    if (!body.name || !body.email || !body.phone || !body.interest) {
      return NextResponse.json(
        { error: "Campos obrigatórios faltando" },
        { status: 400 }
      );
    }

    const lead: Lead = {
      ...body,
      id: crypto.randomUUID(),
      createdAt: new Date().toISOString(),
    };

    console.log("Lead recebido:", lead);

    return NextResponse.json(
      { success: true, message: "Lead registrado com sucesso", lead },
      { status: 201 }
    );
  } catch (error) {
    console.error("Erro ao processar lead:", error);
    return NextResponse.json(
      { error: "Erro interno do servidor" },
      { status: 500 }
    );
  }
}
```
